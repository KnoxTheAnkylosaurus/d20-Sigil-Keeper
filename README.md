# d20-Sigil-Keeper
<img src="https://media.tenor.com/zhIZszouG8QAAAAi/line-divider.gif" width="100%" height="2px"/>
An open-source tool for creating, managing, and playing custom tabletop RPG campaigns — built for Dungeon Masters and players who want full control over their campaigns!

### ✨ Features
**🗺️ Campaign & World Building**
- Create fully custom campaigns
- Design your own maps and locations
- Organize story arcs, quests, and lore
- <img src="https://media.tenor.com/zhIZszouG8QAAAAi/line-divider.gif" width="100%" height="2px"/>

**👤 Character & NPC Management**
- Create detailed player characters and NPCs
- Store stats, backstories, and traits
- Easy overview and editing of all entities
- <img src="https://media.tenor.com/zhIZszouG8QAAAAi/line-divider.gif" width="100%" height="2px"/>

**⚔️ Combat System**
- Built-in combat tracker
- Manage action points, status effects, and turns
- Simplify encounters without losing depth
- <img src="https://media.tenor.com/zhIZszouG8QAAAAi/line-divider.gif" width="100%" height="2px"/>

**📝 Session Tracking**
- Write and store session notes
- Generate short recaps of each session
- Share summaries via email or in-app inbox
- <img src="https://media.tenor.com/zhIZszouG8QAAAAi/line-divider.gif" width="100%" height="2px"/>

**🎲 Player Tools**
- Individual dice system for each player
- Personal notepad
- Character management interface
- <img src="https://media.tenor.com/zhIZszouG8QAAAAi/line-divider.gif" width="100%" height="2px"/>

### 🌍 Philosophy
This project is designed to be free and accessible, with a strong focus on creativity and community. The goal is to provide a powerful tool without locking core features behind paywalls. **But its limited to each Account. If you want to create more stuff, im considering to make maby a Upgrade you can one time payment get full access to your Account.** The project may become open-source, encouraging contributions and customization. 
<img src="https://media.tenor.com/zhIZszouG8QAAAAi/line-divider.gif" width="100%" height="2px"/>

**💡 Monetization (Optional / Planned)**
To support development while keeping the core experience free:
- Free usage includes up to 12 characters per user profile
- Additional character slots may be unlocked via optional paid upgrades
